User Interface


Light Leaker features a GUI interface to create and fine-tune the plug-in effects. Below we'll discuss an overview of the filter's user interface to get started quickly.

After successful plug-in installation, to start, load any image in 8-Bit RGB or CMYK mode into Adobe Photoshop or compatibles and launch the Light Leaker plug-in from the following menubar location: Filter > Mister Retro > Light Leaker*.

* The location to access the Light Leaker plug-in may vary from application to application, consult your imaging application user manual for specific instructions how to access plug-in filters.

Once the plug-in has been successfully registered and launched, you'll notice the controls on the left and the preview window on the right. The preview window features zoom controls beneath and OK and Cancel buttons to apply or cancel the effect with the current settings. Once you've adjusted the controls and are satisfied with the results, click the OK button to apply the Light Leaker effect to your image or click Cancel to close the plug-in and return to your host application leaving your original image unaffected.

Resizable Preview Window - Should you wish to increase or decrease the size of the plug-in window, simply click and drag the lower right hand corner to the desired size you'd like to work with. Light Leaker will remember your preferred dialog window size each time it launches and it can be changed at any time. Also note if your computer memory is limited, reduce the GUI window size for fastest plug-in performace.

User Settings - The User Settings dialog allows you to save or apply previously saved settings to the image you're working with. Once you've created a setting you like, click the Save button, name the setting and click OK to save this setting. To recall and apply a setting, simply click on the setting name in the dialog box. To delete a setting, simply click its name in the dialog box and click the Delete button.

Manual Button - You'll notice in the upper right of the GUI are two buttons: Auto and Manual. If you would like to launch this manual at any time, simply click the Manual button or press the F1 button on your keyboard.

Plug-in Notifications - Never worry about making sure you have the latest version of Light Leaker. As we continue to enhance and improve Light Leaker, the plug-in will automatically notify you when new versions are available and direct you where they can be downloaded. When the Download Update alert is visible, simply click it in the upper left to launch the About dialog, then click the Get Updates button to be taken to the download page to acquire the latest version.

About Dialog - Simply click on the Light Leaker title in the upper left to launch the About dialog where you'll see your serial number and name as well as the button to Get Updates.

Auto Random Effect Generator - We know there are lots of effect options to apply to your image but if you just can't decide which ones to use, click on the Auto button located at the top of the plug-in to randomly load effect settings, you'll be amazed how fun and useful it is!

With so many controls to master, we're confident you'll be thrilled with the variety of visual effects you can create with Light Leaker.
