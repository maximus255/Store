Permanent Press 2 Version History


Stay Current with Permanent Press 2 - In order to assure you have the most updated version of the plug-in and presets, the plug-in will automatically notify you of any updates at the top of the interface upon launch.

When the plug-in has been updated, in the upper right corner of the plug-in will display a Download Update notification. Simply click this button to launch the About Plug-in panel and access a link to download the newest updated version of the plug-in. With each plug-in update not only will features improve, but any bugs discovered will have been fixed.

Release v2.0 (11.24.14) - Plug-in was redeveloped from scratch using an enhanced rendering engine and an updated UI platform - New features include:

  • General - Fully scalable plug-in dialog window. New reset options on all tabbed sections now available. Keyboard commands implemented for zoom ratio views for supported Adobe products. Greatly expanded preset settings with over 200 now available. Automation features were added to allow layer export of ink plates and paper effects.

  • Prepress - Created as a new tabbed section from the former Basic tabbed section functions. Preset settings were moved to a new simplified Basic tabbed section for quick access. Paper effect chooser was added to replace Paper Color along with a host of fine-tune controls.

  • Ink - Ink color picker and fine-tune controls were moved to the Prepress section. Ink Texture Effect chooser was added and a host of new fine-tune features were added including Ink Contrast, Ink Offset and Ink Coverage. Ink Transparency was introduced to adjust ink opacity.

  • Plates - Ink Plate Offset introduced a sphere control. Mask Ink Plate Details was introduced to allow user to reduce plate details for overprinting ink effects. Ink Plate Depth was introduced to allow the user to create emboss/deboss ink plate effects.

  • Halftone - Halftone Type was introduced to allow users to choose from Standard Halftone, Speckled and Screened Halftone styles. Halftone Tint was introduced to adjust volume of halftone per ink plate. Halftone Roughness was introduced to allow user to create more authentic roughened halftone looks.

  • Effects - Formerly Press Effects, Edge Type was introduced with 25 unique edge styles to choose from. Edge Darkness was introduced to darken ink plate edges.

Release v1.05 (Mac) (8.12.10) - 64 Bit native version for Photoshop CS5 released - 32 Bit version renamed to 1.0.5 for version control.

Release v1.04 (Mac) (11.10.08) - Offline registration issues that periodically block plug-in from launching have been resolved.

Release v1.03 (Mac) (10.1.08) - 64 Registration issues with users behind firewalls have been resolved.

Release v1.03 (Win) (6.12.10) - 64 bit native version for Photoshop CS4/CS5 has been released.

Release v1.02 (Mac/Win) (8.29.08) - Memory usage was significantly optimized to allow plug-in to support very large images.

Release v1.01 (Mac/Win) (8.28.08) - Rendering speed improvements and memory issues were fixed which included:

  • CMYK Rendering Speed - In CMYK Color Mode, rendering speeds were significantly increased

  • Preview Color Shifting Issue - In CMYK Color Mode, an issue which caused the preview window to render colors incorrectly was repaired.

  • Memory Usage Issue - A minor memory usage was discovered and repaired

  • Limited Memory Issue - Plug-in was updated to degrade gracefully when there was not enough memory available.

Initial Release v1.0 (8.8.08) - First release of Permanent Press plug-in filter.
